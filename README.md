# DoorLock

## 작품 개요

비대면 형태의 택배와 배달의 증가와 1인 가구 증가로 인해 보안과 편의성이 중요해진 상황에서 원격으로 문의 잠금 상태와 보안 상황을 사용자가 쉽게 확인하고 관리할 수 있는 도어락과 앱의 개발
